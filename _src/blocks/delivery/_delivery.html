Доставка

Стоимость доставки по г. Челябинску:

- для физических лиц 150 рублей.

  • Самовывоз

    Для получения оплаченного заказа необходимо предъявить документ, удостоверяющий личность получателя.
    За дополнительной информацией по доставке вы можете обратиться по телефону, указанном в контактах.

  • Для клиентов из других регионов.

    Доставка до терминала в Челябинске - бесплатная.Этот способ может быть выбран если вы находитесь не в Челябинске. При этом способе доставки заказ отправляется авиа-, авто- или железнодорожным транспортом. Доставка ветровых автомобильных стекол осуществляется только по г.Челябинску.
Доставка по Челябинской области, Екатеринбургу, Кургану, Тюмени, Перми - ТК ЛУЧ (от 100 руб)
